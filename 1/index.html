Model ISO/OSI

Model OSI (pełna nazwa ISO Open System Interconnection Reference Model - model odniesienia łączenia systemów otwartych) lub OSI - standard zdwfiniowany przez ISO oraz ITU-T opisujący srtukturę komunikacji sieciowej.

Model ISO OSI RM jest traktowany jako model odniesienia (wzorzec) dla większości rodzin protokolów komunikacyjnech. Podstawowym założeniem modelu jest podział systemów sieciowych na 7 warstw (ang. layers) współpracujących ze sobą w ściśle określony sposób. Został przyjęty przez ISO w 1984 roku a najbardziej interesującym organem jest wspólny komitet powołany przez ISO/IEC, zwany Jonit Technical Committee 1- Information Technology (JTC1). Formalnie dzieli się jeszcze na podkomitety SC.

Dla Internetu sformulowano uproszczony Model TCP/IP, który ma tylko 4 warstwy.
